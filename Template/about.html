About

Minat dalam Cryptocurrency Saya sangat menyukai dunia kripto dan mata uang digital. Saya mengikuti perkembangan terkini dalam dunia blockchain, serta tertarik dengan aspek-aspek seperti teknologi di baliknya, potensi aplikasi, dan dampaknya pada dunia keuangan global.


Pemula dalam HTML dan CSS Saat ini, saya masih dalam tahap pemula dalam pengembangan web menggunakan HTML dan CSS. Meskipun begitu, saya memiliki semangat dan motivasi untuk terus belajar dan mengembangkan keterampilan saya dalam membangun tampilan visual yang menarik dan responsif.


Tujuan dan Harapan Dengan minat dan pengetahuan yang saya miliki, saya berharap dapat menjadi seorang profesional yang kompeten dalam bidang teknologi informasi, terutama dalam pengembangan web dan penerapan teknologi blockchain. Saya percaya bahwa dengan dedikasi dan kerja keras, saya dapat mencapai tujuan tersebut dan memberikan kontribusi positif dalam industri teknologi.
